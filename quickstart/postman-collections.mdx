## Adım 1: Koleksiyonun Eklenmesi
1000pay API’lerini hızlıca test edebilmek için, hazırladığımız Postman koleksiyonuna erişmek üzere aşağıdaki butonu kullanın. Koleksiyon eklendikten sonra tüm servisleri tek bir yerden inceleyebilir ve çalıştırabilirsiniz.

[Postman](https://testmerchant.1000pay.com/) 

Butona tıkladıktan sonra, Postman üzerinde koleksiyonu online olarak aşağıdaki şekilde görüntüleyebilir ve tüm endpoint’lere tek noktadan erişebilirsiniz.

<img
  style={{ borderRadius: '0.5rem' }}
  alt="Sandbox API Anahtarları"
  src="/images/postman-main-page.png"
/>


### Adım 2: API Bilgilerinin Yapılandırılması
Artık API kimlik bilgilerinizi kullanarak servisleri çalıştırabilirsiniz. Online test işlemlerini gerçekleştirebilmek için Postman hesabınıza giriş yapmanız gerekmektedir.

1. İlk olarak Postman hesabınıza giriş yapın.

<img
  style={{ borderRadius: '0.5rem' }}
  alt="Sandbox API Anahtarları"
  src="/images/potman-sign-in.png"
/>

<Note>
Örnek koleksiyonda yer alan 1000pay servislerinin pre-request script bölümünde, API kimlik bilgilerinin tanımlandığı ve otomatik olarak set edildiği bir environment yapılandırması bulunmaktadır. Bu yapı sayesinde istekler çalıştırılmadan önce gerekli parametreler dinamik olarak hazırlanır.
</Note>

2. API kimlik bilgilerini tanımlayabilmek için yeni bir environment oluşturabilir ya da mevcut 1000pay environment’i üzerinden düzenleme yapabilirsiniz.
Yaptığınız değişiklikleri kaydedebilmeniz için Postman, koleksiyonu fork etmenizi isteyecektir.

<img
  style={{ borderRadius: '0.5rem' }}
  alt="Sandbox API Anahtarları"
  src="/images/posman-environment.png"
/>

3. Koleksiyonu hesabınıza ekledikten sonra, API kimlik bilgilerinizi ilgili alanlarda düzenleyin ve değişikliklerinizi kaydedin.

<img
  style={{ borderRadius: '0.5rem' }}
  alt="Sandbox API Anahtarları"
  src="/images/posman-environment-edit.png"
/>


API kimlik bilgilerinizi tanımladıktan sonra, tüm servisleri test ortamında çalıştırabilirsiniz.

Entegrasyonla ilgili sorularınız olması durumunda entegrasyon@1000pay.com adresi üzerinden bizimle iletişime geçebilirsiniz.
