Bu sayfa, Sandbox ortamında hesap oluşturma ve API erişim bilgilerini edinme sürecine dair gerekli tüm temel bilgileri sunar. Buradaki adımlar sayesinde, canlı ortama geçmeden önce entegrasyonunuzu güvenli bir test ortamında hazırlayabilir ve doğrulayabilirsiniz.


## Adım 1: Sandbox Ortamına Kayıt Olun
<Note>
  **Eğer hâlihazırda bir Sandbox hesabınız bulunuyorsa,** hesabınıza giriş yapmak için [burayı kullanabilirsiniz.](https://testmerchant.1000pay.com/) 
</Note>

Eğer henüz bir Sandbox hesabınız yok ise  [www.1000pay.com](https://www.1000pay.com/kurumsal/sanal-pos/) üzerinden başvuruda bulunabilirsiniz.

Sandbox hesabı; tüm test süreçlerinizi tek bir panel üzerinden yönetmenize olanak tanır. Her hesap, benzersiz bir **merchantId** ile birlikte, entegrasyonlarda kullanılmak üzere **API KEY** ve **SECRET KEY** olmak üzere iki ayrı kimlik bilgisiyle tanımlanır.

| Parametre Adı | Parametre Değeri                           |
| -------- | ------------------------------------- |
| publicKey     | YOUR_PUBLIC_KEY                 |
| privateKey      | YOUR_PRIVATE_KEY                      |
| merchantNumber   | xxxxxxxxxxx |


## Adım 2: Kimlik Bilgilerinizi Edinin
1000pay servislerinde herhangi bir işlem gerçekleştirmeden önce kimlik doğrulama yapılması zorunludur. Gerekli kimlik bilgilerine erişmek için aşağıdaki adımları izleyebilirsiniz:



<Steps>
  <Step>
    Sandbox hesabınıza giriş yapın.
  </Step>
  <Step>
    Sol menüden Bilgilerim > Pos ve Entegrasyon Bilgilerim yolunu izleyin.
  </Step>
  <Step>
    Görüntüle butonuna tıkladığınızda, 1000pay entegrasyonunda kullanılacak kimlik bilgilerine erişebilirsiniz.
  </Step>
</Steps>

## Test Kartları

Sandbox ortamında test yapmak için aşağıdaki test kartlarını kullanabilirsiniz:

- **Başarılı Ödeme**: `4111 1111 1111 1111`
- **Başarısız Ödeme**: `4000 0000 0000 0002`
- **3D Secure**: `4000 0000 0000 3220`

## API Endpoint'leri

Sandbox ortamında kullanabileceğiniz API endpoint'leri:

- **Test URL**: `https://sandbox.api.1000pay.com`
- **Webhook URL**: Test webhook'larınızı yapılandırabilirsiniz
