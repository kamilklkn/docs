Sandbox ortamı, 1000pay entegrasyonunuzu test etmeniz için güvenli bir test ortamı sağlar.

## Sandbox'a Erişim

Sandbox ortamına erişmek için [dashboard](https://dashboard.mintlify.com) üzerinden hesabınıza giriş yapın.

## Test Kartları

Sandbox ortamında test yapmak için aşağıdaki test kartlarını kullanabilirsiniz:

- **Başarılı Ödeme**: `4111 1111 1111 1111`
- **Başarısız Ödeme**: `4000 0000 0000 0002`
- **3D Secure**: `4000 0000 0000 3220`

## API Endpoint'leri

Sandbox ortamında kullanabileceğiniz API endpoint'leri:

- **Test URL**: `https://sandbox.api.1000pay.com`
- **Webhook URL**: Test webhook'larınızı yapılandırabilirsiniz
